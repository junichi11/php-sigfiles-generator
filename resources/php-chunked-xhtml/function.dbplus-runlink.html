dbplus_runlink

(PHP 4 <= 4.3.0, PECL dbplus >= 0.9)

dbplus_runlink — Remove relation from filesystem

Description

dbplus_runlink ( resource $relation ) : int

dbplus_runlink() will close and remove the relation.

Parameters

relation

A relation opened by dbplus_open().

Return Values
