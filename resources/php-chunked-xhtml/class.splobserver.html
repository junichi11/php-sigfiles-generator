The SplObserver interface

(PHP 5 >= 5.1.0, PHP 7)

Introduction

The SplObserver interface is used alongside SplSubject to implement the Observer Design Pattern.

Interface synopsis

SplObserver {
/* Methods */
abstract public update ( SplSubject $subject ) : void
}
