OAuth::enableDebug

(PECL OAuth >= 0.99.3)

OAuth::enableDebug — Turn on verbose debugging

Description

public OAuth::enableDebug ( ) : bool

Turns on verbose request information useful for debugging, the debug information is stored in the debugInfo member. Alternatively, the debug member can be set to a non-FALSE value to turn debug on.

Parameters

This function has no parameters.

Return Values

TRUE

Changelog

Version Description
PECL oauth 0.99.8 The debug and debugInfo members were added
