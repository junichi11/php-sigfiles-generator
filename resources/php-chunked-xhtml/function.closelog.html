closelog

(PHP 4, PHP 5, PHP 7)

closelog — Close connection to system logger

Description

closelog ( ) : bool

closelog() closes the descriptor being used to write to the system logger. The use of closelog() is optional.

Return Values

Returns TRUE on success or FALSE on failure.
