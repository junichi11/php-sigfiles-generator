Imagick::averageImages

(PECL imagick 2.0.0)

Imagick::averageImages — Average a set of images

Warning

This function has been DEPRECATED as of Imagick 3.4.4. Relying on this function is highly discouraged.

Description

Imagick::averageImages ( ) : Imagick

Average a set of images.

Return Values

Returns a new Imagick object on success.

Errors/Exceptions

Throws ImagickException on error.
