Imagick::getImageMatteColor

(PECL imagick 2.0.0)

Imagick::getImageMatteColor — Returns the image matte color

Warning

This function has been DEPRECATED as of Imagick 3.4.4. Relying on this function is highly discouraged.

Description

Imagick::getImageMatteColor ( ) : ImagickPixel

Returns the image matte color.

Return Values

Returns ImagickPixel object on success.

Errors/Exceptions

Throws ImagickException on error.
