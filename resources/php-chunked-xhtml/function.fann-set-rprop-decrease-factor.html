fann_set_rprop_decrease_factor

(PECL fann >= 1.0.0)

fann_set_rprop_decrease_factor — Sets the decrease factor used during RPROP training

Description

fann_set_rprop_decrease_factor ( resource $ann , float $rprop_decrease_factor ) : bool

Sets the decrease factor used during RPROP training.

Parameters

ann

Neural network resource.

rprop_decrease_factor

The decrease factor.

Return Values

Returns TRUE on success, or FALSE otherwise.
