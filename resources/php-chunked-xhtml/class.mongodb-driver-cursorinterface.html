The MongoDB\Driver\CursorInterface interface

(mongodb >=1.6.0)

Introduction

This interface is implemented by MongoDB\Driver\Cursor but may also be used for type-hinting and userland classes.

Class synopsis

MongoDB\Driver\CursorInterface implements Traversable {
/* Methods */
abstract public getId ( ) : MongoDB\Driver\CursorId
abstract public getServer ( ) : MongoDB\Driver\Server
abstract public isDead ( ) : bool
abstract public setTypeMap ( array $typemap ) : void
abstract public toArray ( ) : array
}
