OCICollection::free

(PHP 5, PHP 7, PECL OCI8 >= 1.1.0)

OCICollection::free — Frees the resources associated with the collection object

Description

OCICollection::free ( ) : bool

Frees the resources associated with the collection object.

Return Values

Returns TRUE on success or FALSE on failure.

Changelog

Version Description
8.0.0, PECL OCI8 3.0.0 The OCI-Collection class was renamed to OCICollection to align with PHP naming standards.
