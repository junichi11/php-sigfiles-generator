zip_entry_compressedsize

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PECL zip >= 1.0.0)

zip_entry_compressedsize — Retrieve the compressed size of a directory entry

Description

zip_entry_compressedsize ( resource $zip_entry ) : int

Returns the compressed size of the specified directory entry.

Parameters

zip_entry

A directory entry returned by zip_read().

Return Values

The compressed size.

Changelog

Version Description
8.0.0 This function is deprecated in favor of the Object API, see ZipArchive::statIndex().
