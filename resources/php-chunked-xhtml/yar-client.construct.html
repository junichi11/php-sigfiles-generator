Yar_Client::__construct

(PECL yar >= 1.0.0)

Yar_Client::__construct — Create a client

Description

final public Yar_Client::__construct ( string $url [, array $options ] )

Create a Yar_Client to a Yar_Server.

Parameters

url

Yar Server URL.

Return Values

Yar_Client instance.

Examples

Example #1 Yar_Client::__construct() example
