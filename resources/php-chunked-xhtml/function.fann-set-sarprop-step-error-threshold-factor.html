fann_set_sarprop_step_error_threshold_factor

(PECL fann >= 1.0.0)

fann_set_sarprop_step_error_threshold_factor — Sets the sarprop step error threshold factor

Description

fann_set_sarprop_step_error_threshold_factor(resource $ann, float $sarprop_step_error_threshold_factor): bool

Sets the sarprop step error threshold factor.

Parameters

ann

Neural network resource.

sarprop_step_error_threshold_factor

The sarprop step error threshold factor.

Return Values

Returns true on success, or false otherwise.

Notes

Note:

This function is only available if the fann extension has been build against libfann >= 2.2.
