libxml_clear_errors

(PHP 5 >= 5.1.0, PHP 7)

libxml_clear_errors — Clear libxml error buffer

Description

libxml_clear_errors ( ) : void

libxml_clear_errors() clears the libxml error buffer.

Return Values

No value is returned.
