fann_create_train

(PECL fann >= 1.0.0)

fann_create_train — Creates an empty training data struct

Description

fann_create_train ( int $num_data , int $num_input , int $num_output ) : resource

Creates an empty training data struct.

Parameters

num_data

The number of training data

num_input

The number of inputs per training data

num_output

The number of ouputs per training data

Return Values

Returns a train data resource on success, or FALSE on error.

Notes

Note:

This function is only available if the fann extension has been build against libfann >= 2.2.

  • fann_save_train() - Save the training structure to a file
