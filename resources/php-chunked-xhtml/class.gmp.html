The GMP class

(PHP 5 >= 5.6.0, PHP 7)

Introduction

A GMP number. These objects support overloaded arithmetic, bitwise and comparison operators.

Note:

No object oriented interface is provided to manipulate GMP objects. Please use the procedural GMP API.
